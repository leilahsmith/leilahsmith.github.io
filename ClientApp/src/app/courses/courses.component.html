COURSE CERTIFICATIONS


SOLOLEARN: REACT + REDUX
Earners of this course certificate have demonstrated understanding of core concepts of the react and redux libraries, including JSX, Virtual DOM, Components, hooks, reducers, state management, event-handling, and lifescycle methods.

META: INTRODUCTION TO FRONT-END DEVELOPMENT
This certificate is earned by completing the first course of the Meta Front-End Developer Professional Certificate.

Coursera/Meta: Programming with JavaScript
Earners of this course certificate have demonstrated understanding of basic concepts of web development with JavaScript; working with functions, objects, arrays, variables, data types, the HTML DOM, and much more. Students learned how to use JavaScript and discover interactive possibilities with modern JavaScript technologies. Finally, they learned about the practice of testing code and how to write a unit test using Jest.

Coursera/Meta: Version Control
Earners of this course certificate have demonstrated understanding of basic concepts of implementing Version Control systems, navigating and configuring using the command line, creating and using a github repository, as well as managing code revisions.

Accenture/Forage: Developer Program
This virtual internship experience allows earners of this certificate to work on realistic tasks that developers and technical consultants usually do on real client projects. Earners develop skills in areas such as architecture, requirements, programming, artificial intelligence, testing, cyber security, and application security.
